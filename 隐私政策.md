# 隐私政策

版本：1.0  ·  生效日期：2025-12-05

本隐私政策适用于桌面应用 `bpsys-wpf-idvevent`（WPF）。我们致力于仅收集实现功能所必需的最少信息，并将绝大部分数据保留在您的本地设备上。

## 我们收集的信息
- 账户信息（可选）：当您使用 ASG 赛事接口时，您可在应用中保存并使用 `ASG` 账户邮箱与密码。这些凭据仅用于连接第三方服务 `api.idvevent.cn` 并发起查询/更新请求。
  - 存储位置：`%APPDATA%/bpsys-wpf-idvevent/Config.json`
  - 加密情况：当前以明文字段存储于本地配置文件（未进行额外加密）。
- 应用日志：为定位问题而记录的运行日志与错误信息。
  - 存储位置：`%APPDATA%/bpsys-wpf-idvevent/Log`
  - 日志不包含您的 ASG 密码等敏感内容的刻意采集。
- 本地设置与自定义资源：您的 UI 个性化配置（字体、颜色、图片资源路径）、前台窗口元素位置、OCR 模型镜像与规格等，均保存在本地。
  - 存储位置：`%APPDATA%/bpsys-wpf-idvevent/Config.json` 与同目录下的 `*Config-*.json`、`CustomUi` 子目录。
- 网络资源下载：应用会在您触发时下载 OCR 模型，以及在启动或设置页执行更新检查与下载安装包。这些下载不包含您的个人信息。
- Cookie/令牌：与 `api.idvevent.cn` 交互时会使用会话 Cookie 或访问令牌以维持登录状态；令牌会设置在请求头中用于鉴权。

## 我们如何使用信息
- 身份验证与会话维持：使用您输入的 ASG 账户邮箱与密码登录第三方服务，获取访问令牌以调用赛事查询与比分更新接口。
- 赛事功能：发起赛事搜索、赛程查询、队伍信息获取、比分更新等请求（仅在您明确操作时进行）。
- 应用更新：调用 GitHub Releases API 检查新版本，按需下载并安装更新，不涉及您的个人信息。
- OCR 模型下载：从官方模型源下载所选 OCR 模型以启用识别功能，不收集识别内容到我们的服务器。
- 本地个性化与持久化：保存您在应用中的界面与元素位置偏好，以提升使用体验。

## 信息共享与第三方服务
- ASG 服务（`api.idvevent.cn`）：当您选择登录并使用赛事接口时，我们仅向该服务发送实现功能所必需的数据（如查询参数、比分更新内容）。该服务由第三方运营，适用其自身的隐私政策与条款。
- GitHub（更新检查与下载）：我们使用 GitHub 官方 API 获取发布信息并下载安装包，不会向 GitHub发送可识别您个人身份的数据。
- OCR 模型托管方：OCR 模型由 `Sdcb.PaddleOCR.Models.Online` 提供并托管于官方源，下载时不上传您的个人信息。
- 我们不会出售、出租或以其他方式向第三方披露您的个人信息，除非法律法规要求或为履行您选择使用的第三方功能所必需。

## Cookie 与本地存储
- Cookie：用于与 ASG 服务的会话维持，由该服务的返回控制。应用不将 Cookie 主动持久化到独立文件。
- 本地配置：设置与自定义资源存储在您的用户目录中，仅用于本机功能，不会自动上传。

## 数据安全
- 存储位置：配置与日志文件存放在您的用户目录下（如 `AppData`）。该目录受操作系统的常规访问控制保护。
- 保护措施：我们采用最小化采集、令牌鉴权与标准 HTTPS 通信。当前本地 `Config.json` 未做额外加密处理，请妥善保护您的设备与账户密码（建议仅在需要时保存 ASG 凭据）。
- 漏洞响应：如发现安全问题，您可通过下方联系方式反馈，我们将尽快处理。

## 您的权利
- 访问与更正：您可在设置页修改 ASG 账户、界面样式与其他本地配置。
- 删除与撤回：
  - 在设置页执行“重置配置”以清除多数个性化数据；
  - 手动删除目录 `
    `%APPDATA%/bpsys-wpf-idvevent`
    ` 以移除配置、前台位置与日志；
  - 不登录 ASG 即可停止与赛事服务的数据交互；
  - OCR 下载与应用更新仅在您操作或默认检查时进行，若不需要可不触发下载与安装（启动检查不涉及个人信息）。

## 数据保留
- 本地数据：配置、UI 资源与日志保留在您的设备上，直至您删除或重置。
- 远端数据：通过 ASG 接口提交的赛事与比分数据由该第三方服务保留与管理。

## 跨境传输
- 当访问 GitHub API/下载更新或从官方源下载 OCR 模型时，通信可能经由境外服务器。该等通信仅涉及公开版本信息与模型文件，不包含您的个人身份信息。

## 未成年人保护
- 本软件为通用桌面工具，并非面向未成年人设计。若您为未成年人，请在监护人指导下使用，并避免输入不必要的个人信息。

## 政策变更
- 我们可能因功能调整或法律法规变化更新本政策。重大变更将通过版本发布说明告知。更新后的政策自发布之日起生效。
