Part C: Help content sections

The help dialog should be structured and easy to scan.

Include these sections:

1. Basic workflow
- choose Window and Canvas from top selectors.
- add controls from Add Control.
- select controls from canvas or layer panel.
- edit properties from PropertyGrid.
- save layout.
- validate layout.
- reset/reload layout.

2. Canvas navigation
- mouse wheel / scrollbars to navigate.
- right mouse drag pans the canvas.
- hold Space + left mouse drag pans the canvas.
- Ctrl + mouse wheel zooms.
- bottom Zoom ComboBox accepts 100, 100%, 1x, 1.25x.
- Slider changes zoom.
- Fit button fits the canvas to current viewport.

3. Selecting and editing controls
- click control on canvas to select.
- click item in layer panel to select.
- drag selected control on canvas to move.
- resize with handles.
- selected control name label appears above selection.
- transparent/empty controls may be selected through hitbox or layer panel.

4. Keyboard shortcuts
Include at least:
- Ctrl+S: Save
- Ctrl+C: Copy selected control
- Ctrl+V: Paste copied control
- Delete: Delete selected control
- Ctrl+Z: Undo
- Ctrl+Y: Redo
- Ctrl+Shift+Z: Redo
- Ctrl + Mouse Wheel: Zoom
- Shift: temporarily enable snapping while dragging/resizing
- Space + Left Drag: Pan canvas
- Right Drag: Pan canvas

If the editor supports arrow-key movement, inspect current code and include exact behavior:
- Arrow keys move selected control.
- Ctrl / Shift / Alt modifiers change step size if implemented.
Only document these if actually implemented.

5. Snap behavior
- Snap switch enables snapping.
- Holding Shift temporarily enables snapping without toggling the switch.
- Snap affects move/resize operations.
- Snap does not directly change stored values unless operation is committed.

6. Layer panel
- controls are grouped by ZIndex.
- higher ZIndex is visually above lower ZIndex.
- drag within same layer to change same-ZIndex order.
- drag into another layer to change ZIndex.
- drag to top edge to create new top layer.
- drag to bottom edge to create new bottom layer.
- near top/bottom edge auto-scrolls.
- filtering disables layer reorder; clear filter to reorder.
- linked overlay controls are not independently editable/draggable.

7. PropertyGrid
- Name and BindingPath require Enter or apply button to commit.
- invalid values keep the text and show red border.
- ColorPicker updates Hex text; apply button commits.
- ComboBox properties commit raw values while displaying localized names.
- Binding browser helps choose valid BindingPath.
- Resource browser helps choose bpui/resource URI.
- BorderedImage has separate Border/Image properties.
- BorderedImage resize target can switch between outer border and inner image if available.

8. Add/delete/copy/paste
- Add Control inserts built-in/plugin controls.
- copied control gets unique numbered name on paste.
- paste offsets the new control.
- runtime-critical/generated overlays may not be copyable/deletable.
- Undo/Redo can restore add/delete/paste/reorder operations.

9. Canvas/window settings
- Canvas Width/Height edit canvas design size.
- Background image supports bpui resource URI.
- AllowTransparency is a window-level option and may require restart.
- Restart prompt should ask to save changes.

10. Validation
- status button opens validation details.
- errors may block saving/import/export where appropriate.
- warnings do not always block but should be reviewed.
- missing plugin controls are shown as warnings in Designer.

11. Plugin controls
- plugin controls appear in Add Control when plugin is installed.
- plugin controls behave like built-in controls in Designer.
- missing plugin controls show placeholder in Designer.
- runtime skips missing plugin controls instead of crashing.
- .bpui packages declare required plugins.
- importing a package with missing plugins will guide install/update or allow force import that removes missing controls.

12. Layout packages
- export writes layout package.
- .bpui does not include plugin binaries.
- import checks plugin dependencies.
- force import removes controls that depend on missing plugins.
- original package file remains unchanged.

13. Tips / troubleshooting
- if a control cannot be selected visually, use the layer panel.
- if drag reorder is unavailable, clear search filter.
- if a property appears invalid, check validation details.
- if plugin control is missing, install/enable the plugin and restart if required.
- if Fit makes labels too small, selected label should still stay readable after recent polish.
